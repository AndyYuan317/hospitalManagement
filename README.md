
# dispensary
东莞理工学院城市学院-医务室医疗管理系统<br/>
项目描述：根据我校医务所提出的要求，量身打造一套完全符合学生日常生活健康登记，医生就诊，药房配药，药房入库出库、人事管理、财务统计和菜单权限管理等功能模块<br/>
项目技术：开源框架rapid-framework（基于SSH）+Tomcat7+MySQL+Linux+Myeclipse2014
